Chrome Companion

Help

Getting Started

  • Drag the companion anywhere on the page — she remembers her position.
  • Click the companion to open the chat panel.
  • Configure your LLM provider in Settings (⚙ button) to start chatting.

Chat Commands

Command Description
/help Show available commands in chat
/summarize Summarize the current webpage via LLM
/set-emotion <name> Change sprite — neutral, happy, blush, angry
/clear Clear chat history and conversation context
/settings Open the settings page

Supported LLM Providers

Provider Models
OpenAI gpt-4o-mini, gpt-4o, gpt-4.1-nano, gpt-4.1-mini, gpt-5.2, gpt-5-mini, gpt-5-nano
Anthropic claude-sonnet-4, claude-haiku-4
Google Gemini gemini-3.1-flash-lite, gemini-3.1-pro, gemini-3-flash

Features

  • Draggable — position persists across all sites
  • Emotions — companion swaps between idle sprites and reacts on hover
  • Speech bubble — typewriter animation with speaking sprite cycling
  • Chat panel — full conversation with LLM, animated responses, retro pixel-art style
  • Shadow DOM — styles never leak in or out of the host page
  • SPA resilient — auto re-injects if the page removes the companion

Tips

  • The companion shows happy face when you hover over her.
  • She randomly switches between neutral and idle sprites every 10–15s.
  • She cycles speaking sprites while typing out responses.
  • Conversation history resets on page navigation. Use /clear to reset manually.
  • Use /summarize to get a quick summary of the current page.
  • You can customize the system prompt in settings to change her personality.
