Chrome Companion

Help

Getting Started

  • Drag the companion anywhere on the page — she remembers her position.
  • Click the companion to open the chat panel.
  • Configure your LLM provider in Settings (⚙ button) to start chatting.

Features

  • Draggable companion — drag Riko anywhere on the page, position persists across sites
  • Chat panel — full conversation with LLM, animated responses, retro pixel-art style
  • Speech bubble — typewriter animation with speaking sprite cycling
  • Chat commands — /summarize, /clear, /settings, /stats, /help
  • Social Detox — Riko nags you every 10–15s on greylisted sites to keep you focused
  • Fully customizable — personality, settings, and behavior all configurable
  • Shadow DOM — styles never leak in or out of the host page
  • SPA resilient — auto re-injects if the page removes the companion

Chat Commands

Command Description
/help Show available commands in chat
/summarize Summarize the current webpage via LLM
/clear Clear chat history and conversation context
/settings Open the settings page
/stats Show your top browsing stats (roasted by Riko if LLM is connected)

Settings Tips

  • API Key — you need your own key from the provider. Keys are stored locally and never sent anywhere except the provider's API.
  • Model — smaller/nano models are faster and cheaper. Larger models give better responses.
  • System Prompt — customize Riko's personality. Be descriptive — she'll match whatever vibe you set.
  • Summarize Max Chars — controls how much page text is sent to the LLM for /summarize. Lower = faster/cheaper, higher = more complete.
  • Web Detox — enable the toggle and add sites (one per line) to have Riko nag you when you browse them. Great for staying off social media during work hours.
  • Settings save automatically — no save button needed.
  • Right-click the extension icon → Options to open settings anytime.

Tips & Tricks

  • Conversation history resets on page navigation. Use /clear to reset manually.
  • Detox nag bubbles are suppressed while the chat panel is open.

Privacy

  • No analytics, tracking, or telemetry of any kind.
  • All settings and position data are stored locally on your device.
  • Chat messages are sent directly to your configured LLM provider using your own API key — no intermediary servers.
  • Conversation history is held in memory only and is cleared on page navigation.
